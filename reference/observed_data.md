# Fig wasp community data from Aung et al. 2022

Community composition of fig wasps associated with \*Ficus racemosa\* in
Xishuangbanna.

## Usage

``` r
observed_data
```

## Format

A data frame with \`n\` rows and \`m\` columns:

- Ceratosolen_sp:

  Number of emergent individuals of Ceratosolen_sp

- Sycophaga_testacea:

  Number of emergent individuals of Sycophaga_testacea

- Apocrypta_sp:

  Number of emergent individuals of Apocrypta_sp

- Sycophaga_mayri:

  Number of emergent individuals of Sycophaga_mayri

- Sycophaga_agraensis:

  Number of emergent individuals of Sycophaga_agraensis

- Apocrypta_westwoodi:

  Number of emergent individuals of Apocrypta_westwoodi

## Source

Aung et al. (2022), \[https://doi.org/10.1111/1365-2656.13701\]
